## Run locally
1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Open http://localhost:3000 and go to /login

## Deploy to Vercel
1. Push project to GitHub (upload files to repo root)
2. Import repository into Vercel (Framework auto-detected as Next.js)

This demo uses simulated AI replies. To enable real AI, create an API route and add your OpenAI key in Vercel environment variables.
